# Clients

## Viewing Clients

The **Clients** page shows all your client records. Use the search bar to find clients by name or contact details.

## Adding a Client

1. Click **New Client**
2. Fill in the client's name, email, address, and any other details
3. Click **Save**

## Editing a Client

Open a client's detail page to update their information. Changes are saved to the local database immediately.

## Client-Invoice Relationship

Each invoice is linked to a client. When creating an invoice, you select the client from a dropdown populated from your client list. From a client's detail page, you can see all invoices associated with that client.

## Pricing Tier

Each client can be assigned a **pricing tier**. When a tier is set, line items added from the catalog will use the tier-specific rate instead of the default rate. This is useful for offering different price levels to different client segments.

## Payer Linking

A client can have a linked **payer** (Bill-To party). When you create an invoice or estimate for that client, the payer fields are pre-populated automatically. You can still override payer details on each document. Manage payers from the **Settings** page.

## Bulk Actions

Select multiple clients from the list to perform bulk operations.

## Change History

All client changes are recorded in the audit log. View the change history from the client detail page to see timestamped modifications with field-level diffs.
