# src/lib/components/

Svelte UI components grouped by domain.

## Directories

- `catalog/` — Catalog item management (autocomplete, browse modal, form)
- `client/` — Client management (card, form, select dropdown)
- `csv/` — CSV import/export UI (toolbar, preview modal)
- `import/` — Multi-step import wizard (file select, column mapping, diff preview)
- `invoice/` — Invoice management (card, form, line item row)
- `layout/` — App shell, navigation, file gate
- `payer/` — Payer form
- `pwa/` — PWA reload prompt
- `shared/` — Reusable generic UI components (Button, Modal, etc.)
