# components/shared/

Reusable generic UI components used across the app.

- `Button.svelte` — Styled button with variants
- `Modal.svelte` — Generic modal dialog
- `ConfirmDialog.svelte` — Confirmation prompt modal
- `SearchInput.svelte` — Search input with icon
- `EmptyState.svelte` — Placeholder for empty lists
- `StatusBadge.svelte` — Colored badge for invoice status
- `BulkActionBar.svelte` — Toolbar for bulk selection actions
- `KeyValueEditor.svelte` — Editable key-value pair list
- `LogoUploader.svelte` — Image upload for business logo
