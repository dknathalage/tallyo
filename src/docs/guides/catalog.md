# Catalog

The catalog is your library of products and services. Catalog items can be quickly added to invoices via autocomplete.

## Viewing the Catalog

The **Catalog** page lists all your items. Use search to find items by name or description.

## Adding a Catalog Item

1. Click **New Item**
2. Enter the item name, description, and default unit price
3. Click **Save**

## Using Catalog Items in Invoices

When adding a line item to an invoice:

- Start typing in the description field to search the catalog
- Select an item from the autocomplete suggestions
- The description and unit price are pre-filled from the catalog
- You can override the values for the specific invoice

You can also use the **Browse** button to open a modal and pick items from the full catalog.

## Rate Tiers

Catalog items can have tiered pricing for quantity-based discounts. Configure rate tiers from the catalog item detail page.

## Import & Export

Use the import/export bar on the catalog page to bulk import items from CSV or Excel files, or export your catalog to CSV. See the [Import & Export guide](./import-export) for details.
