# src/docs/

VitePress documentation site source. Builds to `.vitepress/dist/`, deployed at `/invoices/docs/`.

## Files

- `index.md` — Landing page (hero layout)
- `getting-started.md` — Setup and first steps
- `features.md` — Feature overview
- `architecture.md` — Tech stack and project structure

## Directories

- `guides/` — Per-feature user guides (invoices, clients, catalog, import/export, PDF, settings)
- `.vitepress/` — VitePress config and build output
- `public/` — Static assets for docs
