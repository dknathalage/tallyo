# src/

Application source code.

## Files

- `app.html` — HTML document shell
- `app.css` — Global Tailwind styles
- `app.d.ts` — Global TypeScript type definitions
- `pwa.d.ts` — PWA plugin type declarations
- `sql.js.d.ts` — SQL.js library type declarations

## Directories

- `lib/` — Reusable library code (components, db, utils)
- `routes/` — SvelteKit file-based route pages
