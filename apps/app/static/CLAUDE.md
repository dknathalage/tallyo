# static/

Static assets served at the site root. Copied as-is to the build output.

- `favicon.svg` — Browser tab icon
- `robots.txt` — Search engine crawl rules
- `sql-wasm.wasm` — SQL.js WebAssembly binary (SQLite engine)
- `.nojekyll` — Disables Jekyll processing on GitHub Pages
- `icons/` — PWA app icons (SVG, 192px, 512px, Apple touch icon)
